RESUMO DO PROJETO
================================================================================

📋 Sistema de Irrigação Inteligente para Agricultura Digital

🔧 Componentes Implementados:
- ESP32 DevKit v1 (Microcontrolador)
- 3 Botões Push (Sensores NPK: Nitrogênio, Fósforo, Potássio)
- 1 Sensor LDR/Photoresistor (Medição de pH do solo: 0-14)
- 1 Sensor DHT22 (Umidade e temperatura do solo)
- 1 Módulo Relé (Controle da bomba de irrigação)
- 1 LED Azul (Indicador visual de status de irrigação)

⚙️ Funcionalidades:
✅ Monitoramento em tempo real de 5 parâmetros
✅ Lógica de decisão inteligente para irrigação
✅ Controle automático da bomba via relé
✅ Integração com API meteorológica (OpenWeather - Python)
✅ Análise estatística e Machine Learning (R)
✅ Sistema de logging completo

🧠 Lógica de Irrigação:
O sistema IRRIGA quando:
- Umidade do solo < 60% (solo seco)
- E (pH entre 6.0-6.8 OU pelo menos 2 nutrientes NPK presentes)
- E NÃO há previsão de chuva

O sistema NÃO IRRIGA quando:
- Umidade > 80% (solo já úmido)
- Há previsão de chuva ativa
- pH inadequado E falta de nutrientes

🌟 Diferenciais (Ir Além):
✅ EXTRA 1: Integração Python + API OpenWeatherMap
   - Arquivo: python_weather/weather_integration.py
   - Envia previsão de chuva para o ESP32 via Serial

✅ EXTRA 2: Análise Estatística em R
   - Arquivo: r_analysis/irrigation_analysis.R
   - Machine Learning com Random Forest
   - Análises de correlação e visualizações
